# Catálogo de componentes

| Componente | Props | Uso |
|------------|-------|-----|
| Button | variant, size, disabled | Acciones |
| Input | label, error, type | Campos texto |
| Select | label, options, error | Selección única |
| Card | title, children | Contenedor de contenido |
| Table | columns, data, keyExtractor, emptyState (opcional, ReactNode; default: "No hay datos") | Datos tabulares |
| Modal | open, onClose, title | Diálogos |
| Nav | items, activeId | Navegación principal |
| Tabs | tabs, activeTab, onChange | Contenido por pestañas |
| Badge | variant | Estados (success, error, etc.) |
| Skeleton | lines | Loading placeholder |
| Breadcrumb | items | Navegación contextual |
| Toast | message, type | Notificaciones |
| AppHeader | title, navItems, rightContent | Cabecera con nav colapsable |
| AppFooter | links, copyright | Pie de página |
