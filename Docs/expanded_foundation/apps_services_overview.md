# Apps y servicios (referencia)

Resumen de responsabilidades según el pack Expanded Foundation.

## API Gateway

- routing
- auth
- rate limiting
- observability

## Servicios de dominio

- **iam** — identidad y acceso
- **rup** — registro único de proveedores
- **pac** — plan anual de contratación
- **tenders** — procesos de contratación
- **bids** — ofertas
- **evaluation** — evaluación
- **contracts** — contratos
- **documents** — documentos
- **analytics** — analítica

## Packages

- **ui** — componentes de interfaz
- **config** — configuración compartida
- **shared-types** — tipos compartidos frontend/backend
- **api-client** — cliente de API
