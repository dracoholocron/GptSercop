# Frontend Guidelines

## Stack

- Next.js
- React
- TypeScript
- Tailwind
- TanStack Query
- Table (componente del design-system)

## Reglas

- layouts consistentes
- formularios con validación (cliente: email con regex, RUC 13 dígitos donde aplique; atributos required y type="email")
- tablas con filtros y exportación (ej. export CSV en admin/usuarios)
- accesibilidad WCAG 2.1 AA
