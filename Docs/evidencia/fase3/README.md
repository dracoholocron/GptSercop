# Evidencia Fase 3 – Portal entidad: Reportes

## Contenido implementado

- **/reportes**: Página de reportes con resumen de PAC y procesos de la entidad; tabla de últimos procesos con enlace a ofertas. Menú del portal entidad actualizado con "Reportes".

## Cómo reproducir

1. API en marcha; portal entidad: `npm run dev:entity-portal` (puerto 3003).
2. Login con usuario de entidad (ej. admin@mec.gob.ec, rol entity).
3. Navegar a Reportes en el menú.

## Capturas recomendadas

- Pantalla Reportes con resumen (PAC, procesos) y tabla de últimos procesos.
