# Archivo histórico de documentación

Esta carpeta contiene artefactos históricos que se conservan como referencia, pero que no forman parte del flujo activo de implementación.

## Criterios de archivo

- Documentos de revisión puntual o transcript de sesiones pasadas.
- Material útil para trazabilidad, pero no necesario para operación diaria.
- Contenido que no debe eliminarse por contexto histórico/auditoría.

## Nota

La documentación vigente para implementación y operación sigue en `Docs/README.md` y en `Docs/integration/`.
