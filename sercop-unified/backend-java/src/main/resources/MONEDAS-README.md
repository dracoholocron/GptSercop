## Uso en el Sistema

### Endpoints Disponibles

```bash
# Obtener todas las monedas
GET /api/monedas/queries

# Obtener moneda por código
GET /api/monedas/queries/codigo/{codigo}

# Obtener moneda por ID
GET /api/monedas/queries/{id}

# Obtener monedas activas
GET /api/monedas/queries/active

# Buscar monedas
GET /api/monedas/queries/search?query={query}
```

### Ejemplos de Uso

```bash
# Obtener información del Euro
curl http://localhost:8080/api/monedas/queries/codigo/EUR

# Obtener todas las monedas activas
curl http://localhost:8080/api/monedas/queries/active

# Buscar monedas que contengan "Peso"
curl http://localhost:8080/api/monedas/queries/search?query=Peso
```

## Campos de la Tabla

| Campo | Tipo | Descripción |
|-------|------|-------------|
| **id** | BIGINT | Identificador único |
| **codigo** | VARCHAR(3) | Código ISO 4217 (único) |
| **nombre** | VARCHAR(100) | Nombre completo de la moneda |
| **simbolo** | VARCHAR(10) | Símbolo de la moneda |
| **activo** | BIT | Estado (1=activo, 0=inactivo) |
| **version** | BIGINT | Versión para Event Sourcing |
| **createdAt** | DATETIME | Fecha de creación |
| **createdBy** | VARCHAR(255) | Usuario que creó |
| **updatedAt** | DATETIME | Fecha de actualización |
| **updatedBy** | VARCHAR(255) | Usuario que actualizó |

## Notas Importantes

1. **Códigos ISO 4217**: Todos los códigos de moneda siguen el estándar internacional ISO 4217
2. **Símbolos Unicode**: Los símbolos de moneda utilizan caracteres Unicode para correcta visualización
3. **Estado Activo**: La mayoría de monedas están activas (activo=1), excepto las criptomonedas que están desactivadas por defecto
4. **Event Sourcing**: Todas las monedas tienen version=0 inicialmente para el sistema de Event Sourcing
5. **Auditoría**: Todos los registros incluyen información de auditoría (createdAt, createdBy)

## Mantenimiento

### Agregar Nueva Moneda

```sql
INSERT INTO moneda_read_model (id, codigo, nombre, simbolo, activo, version, createdAt, createdBy)
VALUES (NEXT_ID, 'XXX', 'Nombre de la Moneda', 'SYM', 1, 0, NOW(), 'ADMIN');
```

### Activar/Desactivar Moneda

```sql
-- Desactivar
UPDATE moneda_read_model SET activo = 0 WHERE codigo = 'XXX';

-- Activar
UPDATE moneda_read_model SET activo = 1 WHERE codigo = 'XXX';
```
