# Contexto del Sistema

La plataforma V2.0 busca modernizar la gestión de contratación pública.

Funciones principales:

- Registro de proveedores (RUP)
- Plan Anual de Contratación (PAC)
- Gestión de procesos de contratación
- Recepción y evaluación de ofertas
- Gestión contractual
- Datos abiertos y analítica

Principios de diseño:

- Arquitectura modular
- API-first
- Datos abiertos
- Seguridad y trazabilidad
- Escalabilidad cloud
